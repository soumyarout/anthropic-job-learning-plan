Your path to Anthropic RL Research Engineer

General learning roadmap — no PhD required

JOB #5115935008
Your edge: You already build with AI APIs, understand enterprise use cases, and think in systems — this is a genuine advantage over pure academics. The gap is depth in ML theory, PyTorch internals, and RL. This roadmap closes exactly that.
PHASE 1 · MONTHS 1–3

Foundation — Math, Python, and ML intuition

⏱ 3 months ⚡ 10–15 hrs/week ★ Solidify the base before building up
∑
Linear Algebra & Calculus for MLVectors, matrices, gradients — the language of neural nets
MUST ▼

Every forward/backward pass is matrix multiplication and chain-rule calculus. You cannot debug models or read papers without this.

  • Course
    MIT 18.06 Linear Algebra — Gilbert Strang (OCW)
    Free lecture videos + problem sets. Watch lectures 1–15 minimum.
  • Video
    3Blue1Brown — Essence of Linear Algebra (YouTube)
    Visual intuition playlist. Watch before Strang, not instead of.
  • Video
    3Blue1Brown — Essence of Calculus (YouTube)
    Covers derivatives, chain rule, gradients intuitively.
  • Book
    Mathematics for Machine Learning (Free PDF)
    Cambridge textbook, free online. Chapters 2–5 cover exactly what you need.
π
Probability & StatisticsDistributions, expectations, Bayesian thinking
MUST ▼

RL is fundamentally probabilistic — policy gradients, value estimation, and reward modelling all rest on probability theory.

  • Course
    Seeing Theory — Brown University (Free, interactive)
    Visualise probability concepts interactively. Great starting point.
  • Course
    MIT 6.041 Probability (OCW)
    Full lecture series, free. Focus on first 12 lectures.
  • Book
    Grinstead & Snell — Introduction to Probability (Free PDF)
    Classic free textbook, very readable.
🐍
Python — Research gradeNumPy, data structures, clean code patterns
MUST ▼

You likely know Python basics already. The upgrade is NumPy fluency (vectorised ops, broadcasting), type hints, and writing readable research code.

  • Course
    CS231n Python/NumPy Tutorial (Stanford, free)
    Written by Stanford. The essential NumPy patterns used in every ML codebase.
  • Practice
    Kaggle Learn — Python + Pandas (Free)
    Hands-on micro-courses with in-browser coding.
  • Repo
    Python Data Science Handbook — Jake VanderPlas (Free on GitHub)
    Full book as Jupyter notebooks. Focus on NumPy and Matplotlib chapters.
📊
Classical ML intuitionRegression, classification, overfitting, cross-validation
KEY ▼

Understanding bias-variance trade-off, regularisation, and evaluation metrics grounds you before deep learning — many RL errors are classical ML errors in disguise.

  • Course
    Andrew Ng — Machine Learning Specialisation (Coursera, audit free)
    The canonical intro. Audit all 3 courses for free. Algorithms course is most relevant.
  • Book
    An Introduction to Statistical Learning (ISLR, Free PDF)
    Stanford textbook, freely downloadable. Chapters 2–4 cover the essentials.
PHASE 2 · MONTHS 4–9

Core ML/DL — Deep learning and PyTorch mastery

⏱ 6 months ⚡ 15–20 hrs/week ★ Build and train real models from scratch
🔥
PyTorch — Deep masteryTensors, autograd, custom training loops, GPU
MUST ▼

Anthropic's codebase is PyTorch. You need to write custom loss functions, implement paper equations in code, and debug GPU memory issues confidently.

  • Course
    Official PyTorch Tutorials (Free)
    Start with "Learn the Basics" → "Introduction to PyTorch" series. Do every tutorial hands-on.
  • Course
    fast.ai Practical Deep Learning (Free)
    Top-down, code-first. Lesson 1–8 in Part 1. Famous for building intuition fast.
  • Video
    Andrej Karpathy — Neural Networks: Zero to Hero (YouTube)
    Build backprop, micrograd, GPT from scratch. Unmissable. Do all 8 videos.
  • Practice
    LLMs from Scratch — Raschka (GitHub, free book)
    Implement a GPT-like model step-by-step. Bridges PyTorch skills to transformer understanding.
🧠
Deep learning theoryCNNs, RNNs, attention, normalisation, optimisers
MUST ▼

You need to understand why Adam works, what batch norm does, and why residual connections matter — to read and implement RL papers that build on these primitives.

  • Course
    Stanford CS231n — CNNs for Visual Recognition (Free slides + notes)
    Best deep learning fundamentals course. Lecture notes are gold even without a video.
  • Course
    DeepLearning.AI Deep Learning Specialisation (audit free)
    Andrew Ng's 5-course series. Audit all for free. Courses 1–4 are core.
  • Book
    Dive Into Deep Learning — d2l.ai (Free, interactive)
    Full textbook with runnable code. Chapters 4–9 cover everything in this skill.
⚙️
Transformers & attention mechanismSelf-attention, multi-head, positional encoding
MUST ▼

All of Anthropic's work is transformer-based. You must understand the architecture deeply enough to modify it, not just call it.

  • Paper
    Attention Is All You Need — Vaswani et al. (2017, free arXiv)
    The original transformer paper. Read it 3 times at different stages of your learning.
  • Video
    3Blue1Brown — Attention in Transformers (YouTube)
    Visual walkthrough of self-attention. Watch before reading the paper.
  • Repo
    nanoGPT — Andrej Karpathy (GitHub)
    ~300 line clean GPT implementation. Read, run, modify. Best learning tool for transformers.
  • Course
    Hugging Face NLP Course (Free)
    Practical transformers using HF libraries. Chapters 1–3 cover transformer architecture hands-on.
🖥️
ML engineering & infrastructureExperiment tracking, distributed training, profiling
KEY ▼

Research engineers at Anthropic own the full stack. You need to manage training runs, track experiments, and diagnose GPU bottlenecks.

  • Course
    Full Stack Deep Learning 2022 (Free)
    Covers MLOps, experiment tracking, deployment. Best resource for the engineering side of ML.
  • Repo
    Weights & Biases Tutorials (Free)
    Learn experiment tracking with W&B — standard tool at most research labs.
  • Book
    PyTorch Distributed Training Guide (Official, free)
    DDP, model parallelism, gradient checkpointing — all used in large-scale RL training.
PHASE 3 · MONTHS 10–18

RL + LLMs — The core of the role

⏱ 9 months ⚡ 20 hrs/week ★ The exact intersection Anthropic works in
🎮
Reinforcement Learning fundamentalsMDPs, Q-learning, policy gradients, value functions
MUST ▼

This is the core job. You need to implement PPO, understand reward shaping, and debug RL training instabilities — all from first principles.

  • Book
    Sutton & Barto — Reinforcement Learning (Free 2nd edition)
    The bible of RL. Read Chapters 1–7 (tabular methods) then 13 (policy gradient). Essential.
  • Course
    DeepMind x UCL — RL Course 2021 (YouTube, free)
    Hado van Hasselt's definitive lecture series. 13 lectures, each 90 min. Do all of them.
  • Course
    OpenAI Spinning Up in Deep RL (Free)
    Explains key algorithms (VPG, PPO, SAC) with clean implementations. Build each one yourself.
  • Repo
    CleanRL — Single-file RL implementations (GitHub)
    Each algorithm in one readable file. PPO, DQN, SAC. Read and re-implement from scratch.
🤖
RLHF & LLM alignment methodsPPO for LLMs, reward modelling, DPO, GRPO
MUST ▼

Anthropic invented RLHF for Claude. This is exactly what the RL team does — training reward models, running PPO on LLMs, and developing next-gen methods like GRPO.

  • Paper
    Training language models to follow instructions with human feedback — InstructGPT (arXiv)
    The original RLHF paper. The foundation of Claude's training pipeline.
  • Paper
    Direct Preference Optimisation (DPO) — Rafailov et al. (arXiv)
    Key alternative to PPO-based RLHF. Understand the trade-offs deeply.
  • Repo
    TRL — Hugging Face (GitHub, free)
    Open-source RLHF library. Study the PPOTrainer and DPOTrainer implementations.
  • Video
    Andrej Karpathy — State of GPT (YouTube)
    Outstanding overview of RLHF pipeline from InstructGPT to ChatGPT. Watch multiple times.
🕵️
Agentic AI & tool useAgents, computer use, multi-step reasoning, tool calling
MUST ▼

The job description specifically mentions computer use and autonomous task execution. Anthropic's RL team builds the systems that enable Claude to use tools and act in the world.

  • Course
    AI Agents in LangGraph — DeepLearning.AI (Free)
    Hands-on agentic patterns. Build agents with tool use, memory, and multi-step planning.
  • Paper
    ReAct: Synergizing Reasoning and Acting in LLMs (arXiv)
    Foundational paper on thought-act-observe loops. Implemented in most agent frameworks.
  • Repo
    Anthropic Cookbook (GitHub, free)
    Official examples of tool use, agentic patterns, computer use with the Claude API. Study every notebook.
  • Paper
    WebArena — Benchmark for Autonomous Agents (arXiv)
    Understand how computer-use agents are evaluated. Referenced in Anthropic's own work.
🔬
LLM fine-tuning & training at scaleLoRA, PEFT, gradient checkpointing, multi-GPU
KEY ▼

Research engineers own the training code. You need to fine-tune LLMs, understand memory trade-offs, and run experiments across GPUs efficiently.

  • Course
    Fine-tuning LLMs — DeepLearning.AI (Free)
    Practical walkthrough of fine-tuning with Lamini. Good entry point.
  • Paper
    LoRA: Low-Rank Adaptation of LLMs (arXiv)
    The parameter-efficient fine-tuning paper. Read and implement it yourself.
  • Repo
    Unsloth — Free, fast LLM fine-tuning (GitHub)
    Run LoRA fine-tuning on free Google Colab GPUs. Practical experience with real models.
🛡️
AI Safety & alignment basicsConstitutional AI, reward hacking, scalable oversight
KEY ▼

Anthropic's entire mission is safe AI. You'll collaborate with alignment researchers — you don't need to be one, but you must understand their goals and vocabulary.

  • Paper
    Constitutional AI — Anthropic (arXiv)
    The paper that describes how Claude is trained. Read it carefully — it IS the job.
  • Course
    AI Safety Fundamentals — Alignment Course (Free)
    8-week curriculum covering reward misspecification, scalable oversight, interpretability basics.
  • Book
    Cold Takes — AI risk writing (Free blog)
    Accessible framing of why alignment matters. Builds the mental models Anthropic operates with.
PHASE 4 · MONTHS 19–24+

Research-ready — papers, projects, and presence

⏱ 6+ months ongoing ⚡ 20+ hrs/week ★ Build proof of work, not just knowledge
📄
Reading & implementing ML papersarXiv, paper → code, ablations, reproducibility
MUST ▼

Research engineers at Anthropic read new papers weekly and implement them. This is a learnable skill — start with reproductions, then move to extensions.

  • Video
    Andrej Karpathy — How I read ML papers (YouTube)
    His exact paper-reading workflow. Follow it religiously.
  • Repo
    Papers With Code (Free)
    Every major ML paper linked to its official code. Pick RL papers and re-implement them.
  • Practice
    arXiv cs.LG — read 3 abstracts/day (Free)
    Daily habit. Scan abstracts, pick 1 per week to read deeply, 1 per month to implement.
🏗️
Build a portfolio of RL projectsOpen-source contributions, GitHub, demos
MUST ▼

Without a PhD or FAANG brand, your GitHub is your CV. You need 3–5 substantial RL projects that show you can implement ideas independently.

  • Practice
    Gymnasium (formerly OpenAI Gym) — Free
    Standard RL environments. Implement PPO, SAC, DQN from scratch and train agents on classic control tasks.
  • Practice
    Project: RLHF pipeline from scratch
    Build a reward model on a small dataset, fine-tune a small LLM with PPO using TRL. Write a blog post explaining it.
  • Practice
    Project: Reproduce a recent Anthropic/DeepMind paper
    Pick a paper from Anthropic's publications page, implement the key result, publish results on GitHub.
  • Repo
    Anthropic Evals (GitHub, free)
    Contribute to open eval sets. Familiarity with how Anthropic evaluates models is a direct signal.
